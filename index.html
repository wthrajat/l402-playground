What is L402?

L402 is an internet-native authentication protocol that leverages the HTTP 402 Payment Required status code combined with Lightning Network payments and Macaroon tokens.

Key Properties

  • No accounts needed — authentication via proof of payment
  • Instant micropayments — pay fractions of a cent per request
  • Machine-friendly — designed for AI agents and automation
  • Privacy-preserving — no personal data required
  • Cryptographically verifiable — stateless server-side validation

Specification

Formalized as bLIP-0026 in the Bitcoin Lightning Improvement Proposals.

Components

Macaroon Bearer token with attenuable caveats
Lightning Invoice BOLT11 payment request for sats
Preimage SHA-256 proof of payment
L402 Token macaroon:preimage sent in Authorization header

Built for learning. No real Lightning payments involved. • l402.org
